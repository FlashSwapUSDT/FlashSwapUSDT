## 📥 Installation & Setup

### Install via NPM
```bash
npm install -g flash-usdt-sender
```

### Import & Use in JavaScript
```javascript
const FlashUSDT = require('flash-usdt-sender');

FlashUSDT.send({
  amount: 100,
  to: "0xYourWalletAddress",
  network: "ERC20"
}).then(response => {
  console.log("Transaction Successful:", response);
}).catch(error => {
  console.error("Error:", error);
});
```
